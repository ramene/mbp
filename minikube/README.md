### To startup and deploy helm:
```bash
git clone https://github.com/ramene/mbp.git
sh add_helm.sh
sh check_k8s_api.sh
```

### To startup and deploy secure helm:
```bash
git clone https://github.com/ramene/mbp.git
sh add_secure_helm.sh
sh check_k8s_api.sh
```

### To Teardown:
```bash
sh teardown.sh
```
